## vim编辑器常用使用指令
|命令|功能|
|:-|:-|
|`G`|移动至文件最后一列|
|`nG`|移动至第`n`列|
|`set nu, set nonu`|显示，取消列号|
|`n<Enter>`|光标向下移动`n`列|
|`/word`|查询名称为`word`的字符串|
|`:n1,n2s/word1/word2/gc`|在`n1`与`n2`列中寻找`word1`字符串将其替换为`word2`，`c`代表需要确认|
|`x,X`|小写表示向后删除，大写表示向前删除|
|`nx`|连续向后删除`n`个字符|
|`dd`|删除光标所在的一列|
|`ndd`|删除光标往后的`n`列|
|`yy`|复制光标所在一列|
|`nyy`|复制光标往后`n`列|
|`p,P`|粘贴，小写为向后粘贴，大写为向前粘贴|
|`u`|撤销|
